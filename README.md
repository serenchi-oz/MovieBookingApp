# 🎬 MovieApp

A full-stack **Movie Management Web Application** built with **ASP.NET Core**, **Angular**, and **SQL Server**.  
This app allows admins to manage movies, users to browse and view details, and integrates **Google Gemini AI** for enhanced features like descriptions and recommendations.

---

## 🌟 Features

### 🧩 Backend - ASP.NET Core
- RESTful **Web API** using **Entity Framework Core (DB First)** approach.
- **Role-Based Authorization** using **JWT (JSON Web Token)**.
- Implements **Repository Pattern** for clean separation of concerns.
- **Dependency Injection (DI)** for maintainability and testability.
- **CRUD operations** for admin panel to manage movies and users.

### 💻 Frontend - Angular
- Built with **Angular Material** for a modern UI.
- **Routing & Navigation** with **lazy-loaded modules** for performance optimization.
- **Auth Guards** to protect routes based on user roles.
- **HTTP Interceptors** for handling API requests and JWT tokens.
- **Reactive Forms** with **inbuilt & custom validation**.
- **Custom Pipes** for data transformation.
- **CSS effects**: Movie posters enlarge on hover for a better UX.
- **Recommended Movies Section** based on user preferences.
- **Google Gemini AI Integration** for movie descriptions and recommendations.
